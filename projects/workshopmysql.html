MySQL database management

This introductory course to MySQL teaches people basic concepts of RMDB systems.

The workshop mainly is about programming with SQL. We put into practive basic queries like SELECT, INSERT and UPDATE but also use GROUP BY and JOIN. Mainly we use PHPMyAdmin as database client and finish the course with a basic example making a website using PHP.
