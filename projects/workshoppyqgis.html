Workshop pyqgis

This introductory course you will learn to use Python in combination with QGIS. The Python API of QGIS pyqgis offers all the possibilities to customize QGIS and to automatize processes.

We also learn how to design and program a basic QGIS plugin in order to expand and enhance the possibilities of QGIS.
