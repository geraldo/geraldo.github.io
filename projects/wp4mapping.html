Wordpress4Mapping

Simple software solution for basic mapping tasks based on Wordpress and Openlayers.

This project was created for mobile use to go out on the street and map POIs.

It uses the following plugins and libraries:

  • ACF for forms.
  • Openlayers for maps.

Custom development includes:

  • Theme mapping based on undescores.
  • Plugin mapping to register fields, taxonomies and CPT poi.
  • Plugin acf-map-field to include map in form.

Client: UOC Nodes, xrcb.cat
