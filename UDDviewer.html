Atlas bithabitat

Tools and metodologies for urban planning.

Bithabitat works on urban planning processes using open data. To bring their outputs online, we developed a set of tools to make this process easy and fast.

It includes plugins for QGIS to use and update open data catalogues, and a web map viewer to show the data online.
